# yahoo-finance-ledger
Downloaded Yahoo Finance Plus data, notes, and structured exports for analysis and reference.

## GTLB exports
- `GTLB_quarterly_valuation_measures_ehm.csv`: raw quarterly valuation measures export.
- `GTLB_quarterly_valuation_measures_ehm_clean.csv`: cleaned numeric fields (thousands separators removed) for easier Excel import.
- `GTLB_quarterly_valuation_measures_ehm_tidy.csv`: long-format (metric/period/value) version for easier filtering and pivoting.
